Responsive Design

Responsive design is the practice of implementing HTML, CSS, and JavaScript so that a page is fully legible, attractive, and has a good user experience across all screen sizes (desktop, laptop, tablet, and mobile). Media queries can be used to set different styling across various screen sizes by setting styling at specific width breakpoints. You can also use percentages and viewports to set responsive styling.

Google Analytics

Google Analytics is a widely used, relevant tool that can help digital marketers understand which pages are being viewed on their site and the analytics connected to those views.

Here are three types of reports that can be useful for your site:

 

  1. The Audience Overview Report shows the main metrics of the site, includign total users, total seessions, pageviews, pages per sessions, average visit duration, and bounce rate of the overall site.
  2. The Queries Report shows the top keywords that users are searching for that lead them to the site, which can be very useful to understand for search engine optimization through keywords on the site.
  3. Site speed is another important consideration for an optimal website. The Site Speed Report shows the site speed page overview including the average page load time, average redirect time, average server response time, average page download time, and more. In general, it's a good guideline is to keep page load time under 2 seconds.
